Introduction

Welcome to the BitDegree API! You can use our API to access BitDegree API endpoints, which can get information on various courses, instructors and course categories in our database.

Currently, we have language bindings in Shell. You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

Authentication

To authorize, use this code:

# With shell, you can just pass the correct header with each request
curl "api_endpoint_here" -H "Authorization: your_api_key"

Make sure to replace your_api_key with your API key.

BitDegree uses API keys to allow access to the API. You can register a new BitDegree API key at our affiliates area.

BitDegree expects for the API key to be included in all API requests to the server in a header that looks like the following:

Authorization: your_api_key

You must replace your_api_key with your personal API key.

Coupon can be used for all courses. 1 api_key can have a maximum of 1000 coupons.

Coupon object attributes

ParameterTypeDescription
idIntegerID of coupon
codeStringUnique code for all platform courses.
amountStringCourse discount in percentage
quantityIntegerHow many coupons can be used for Successful orders. Can be null for unlimited
statusStringCoupon status. enabled, disabled
expires_atTimestampDate, when coupon will expire. Can be null
created_atTimestampCoupon creation date
updated_atTimestampLatest update for this coupon
coupons_usedIntegerSuccessful orders with this coupon

Create a coupon

curl -X POST "https://www.bitdegree.org/api/v1.0/coupon" -H "Authorization: your_api_key" -H "Content-Type: application/json" -d '{"code":"YOUR_CODE", "percentage":10}'

The above command returns JSON structured like this:

{
    "id": 439499,
    "code": "BITDEGREE_COURSES",
    "amount": 30,
    "quantity": 200,
    "status": "enabled",
    "expires_at": "2020-12-31 00:00:00",
    "created_at": "2020-07-23 12:33:30",
    "updated_at": "2020-07-23 12:42:10",
}

This endpoint is used for creating a coupon.
Successful request returns newly created coupon object

POST https://www.bitdegree.org/api/v1.0/coupon

expires_atDateFalseFormat Y-m-d, 2020-01-01.

Errors

BitDegree API uses the following error codes:

Error CodeMeaning
400Bad Request -- Your request is invalid.
401Unauthorized -- Your API key is wrong or missing.
403Forbidden -- Endpoint usage is forbidden for your user. Check message.
404Not Found -- The specified course could not be found.
414Request-URI Too Long -- Request parameters are too long. Maximum length: 190.
429Too Many Requests -- Daily request limit reached.
500Internal Server Error -- We had a problem with our server. Try again later.
503Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
